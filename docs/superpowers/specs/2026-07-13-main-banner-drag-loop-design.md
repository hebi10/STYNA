# 메인 배너 드래그·무한 순환 안정화 설계

## 목표

- 마지막 배너에서 첫 배너로 넘어갈 때 보이는 깜박임을 제거한다.
- 전환 중 추가 입력을 무시해 빠른 연속 클릭이 역방향 장거리 이동으로 보이지 않게 한다.
- 드래그 중 트랙이 포인터의 X축 이동량을 실시간으로 따라가게 한다.
- 포인터를 놓았을 때 시작점에서 48px 미만이면 원위치하고, 이상이면 방향에 따라 한 장만 이동한다.

## 설계

- 순환용 복제 슬라이드는 유지하되 끝점 보정은 `transform`의 `transitionend`에서만 수행한다. 고정 시간 타이머 기반 보정은 제거한다.
- `isAnimating` 상태로 버튼, 자동 전환, 드래그 종료에 의한 새 이동을 잠근다. 전환 종료와 복제 슬라이드의 즉시 위치 보정이 끝난 뒤 잠금을 해제한다.
- 포인터 시작 X와 현재 드래그 X를 분리해 저장하고, 트랙의 기본 퍼센트 이동값에 드래그 픽셀 오프셋을 더한다.
- 48px 미만 드래그는 같은 트랙 인덱스를 다시 적용해 원위치 애니메이션을 실행한다. 48px 이상 드래그는 좌측이면 다음, 우측이면 이전 슬라이드로 이동한다.
- 4px 이상 움직이면 드래그로 판정해 상품 링크 클릭을 방지하되, 거의 움직이지 않은 일반 클릭은 유지한다.

## 검증

- 마지막→첫 번째 이동 후 `transitionend`에서 복제 위치가 실제 첫 번째 위치로 즉시 보정되는지 테스트한다.
- 전환 중 연속 다음 클릭이 한 번만 반영되는지 테스트한다.
- 포인터 이동 중 X축 오프셋이 트랙 스타일에 반영되는지 테스트한다.
- 48px 미만 드래그는 인덱스를 유지하고, 이상 드래그는 한 장 이동하는지 테스트한다.
- 4px 이상 움직인 짧은 드래그가 상품 링크를 열지 않는지 테스트한다.
